내적!
3D는 다 내적부터 시작한다, 구구단같은 느낌

내적이 무엇이느느냐 : 그러게 내적은 뭐하는 새끼임? 

나무위키 피셜 ) 벡터 공간에서 정의된 이중선형 혹은 반쌍형적 함수의 일종. 보통 내적은 벡터의 방향이 얼마나 일치하는지를 알기 위한 용도로 쓰인다.
고등학교에서는 '내적'이라고 부르는 연산은 대학교에서는 '도트곱 (dot product)' 혹은 '스칼라곱 (scalar product)' 이라고 부르는, 
내적의 한 종류이며, 실제로는 무한히 많은 종류의 내적이 있다. 내적의 공리만 만족시키면 구체적인 연산이 어떻든 내적이라 부르기 때문이다. 
하지만 내적을 어떻게 정의하던 유한차원에서 정규직교기저(orthonormal basis) 를 이용하면 도트곱과 같은 형태로 유도할 수 있기에 내적은 본질적으론 도트곱 하나뿐이다.

어.. 간단하게 벡터 공간에서 정의된 함수라는거네?

내적 공식 : a의 길이 * b의 길이 * cos세타(두 벡터 사이의 각) = ax*bx + ay*by + az*bz
쎄타의 범위는 0~180도 안에서 유효하다

코사인 제2법칙을 어떻게어떻게 하면 ax*bx + ay*by + az*bz 일로온다


내적의 활용처들
- 두 벡터의 사이각을 얻기 위해 사용 (사잇값)
- 두 벡터가 가리키는 방향의 차이를 알기 위해서 활용 (차잇값?)
- 두 벡터를 분해하여 수직 및 평행한 백터를 얻기 위해서 활용 (투영)

내적의 활용처의 활용처들(?)

사잇값 - 조명 강도를 조절할 때 (22-14-1) 
차잇값 - 내가 바라보는 시선벡터랑 뭐 대충 어디 벡터랑 내적했을 때 + : 내 앞에 있다, - : 내 뒤에 있다 (그림 22-14-3)
투영 - 직선으로부터 한 점이 얼마나 떨어져있는지 구할 때 (ex - 히트스캔 광선이 구 형태의 뭐시기한테 맞았나 안맞았나 체크할 수 있음, 구의 중심과 직선 사이의 거리가 구의 반지름보다 크면 안맞은거)





외적!
외적이 뭐냐 - 두 벡터 a,b가 있을때 a,b 각각에 대하여 수직인 벡터
외적은 위, 아래로 두개가 있다, 즉 교환법칙이 성립하지 않는다
a x b랑 b x a랑은 전혀 다르다
DirectX에서는 왼손 좌표계를 사용하기 때문에 b x a로 가야한다 

내적은 스칼라, 외적은 벡터가 만들어진다

벡터의 외적 공식 1,2 (3,4는 행렬인데 아직  안했으니까 패스)
외적 공식2 : a X b = (aybz-azby, azbx-axbz, axby-aybx)
외적 공식2에서 부호가 -, 시작이 y라는것만 기억하면 된다

외적은 안 나올 수가 있다 (값이 0,0,0일 수 있다), 언제? 
모든 성분이 동시에 0이 나올 때 -> 두 벡터가 나란하면(평행하면) 외적 안나온다

외적은 교환, 결합법칙이 성립하지 않는다, 분배법칙은 성립한다. (걔들이뭔데이씹덕아)

외적이라는건 면적으로 이해해도 된다? (평행사변형의 면적을 외적벡터의 크기로 구할 수도 있다..)


외적은 어따 써먹느느냐? - 평면의 방정식

평면의 방정식 : ax + by+ cz + d = 0
a,b,c = 법선 벡터 (평면이 바라보는 방향)
d = 원점과 평면 사이의 최소거리

어떤 점을 평면의 방정식에 넣었을 때 0이 나온다면, 평면상에 존재하는 점이다
값이 +가 나오면 평면 앞에 있고, -가 나오면 평면 뒤에 있는것  // ex) a <-ㅣ b  라면 a는 +, b는 -가 나올 것  

천장의 법선벡터 : 0, -1, 0
바닥의 법선벡터 : 0, 1, 0
칠판의 법선벡터 : 0, 0, -1
창문의 법선벡터 : 0, 0, 1
왼쪽 벽의 법선벡터 : 1, 0, 0
오른쪽 벽의 법선벡터 : -1, 0, 0

평면의 방정식은 어따 써먹음? 
- 총을 쐈을때 벽에 맞았음, 근데 그 교점이 벽의 한 점인지 어뜨케 알아? 그럴때 벽의 평면의 방정식에 교점을 넣으면 댄다


아무튼 평면의 법선벡터를 외적을 통해서 구할 수 있다 (d는?)
a * b  = -d (내적인가??) 아씨 놓쳤어
d = -(a*b)
원래 평면의 방정식 변형해서 -(ax + by+ cz) = d 인데 
이걸 또 변형해서 ax + by+ cz 는 사실 내적이니까(맞냐?) 그냥 a*b로 표시인듯
그래서 abc가 외적이라면 d는 내적이다
